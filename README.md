# david-cli
[![NPM](https://nodei.co/npm/david-cli.png?downloads=true&downloadRank=true&stars=true)](https://nodei.co/npm/david-cli/)
## Install
```bash
npm install -g david-cli
``` 
or 
```bash
yarn global add david-cli
``` 

## How to use it ?

```bash
david-cli --manifest='path/to/package.json'
```
